    <div style={S.cmt}>{'/* 2. Add the semantic alias that components consume */'}</div>
    <div><span style={S.key}>--astral-color-text-accent-default</span>{': var(--astral-color-purple-9);'}</div>
    <br />
    <div style={S.cmt}>{'/* 3. Use the semantic token in components — never the primitive */'}</div>
    <div>{'.badge { '}<span style={S.key}>color</span>{': var(--astral-color-text-accent-default); }'}</div>
  </div>
  <div style={S.tip}>
    <strong>Rule:</strong> New primitives should only be added when the value is genuinely new to the system. New semantic tokens should be added when a new design intent (role) exists. When in doubt, re-use an existing semantic token.
  </div>
