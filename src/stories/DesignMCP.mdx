  Spacing       →  var(--astral-units-pos-{0-99}) / var(--astral-units-neg-{1-10})
  Border radius →  var(--astral-border-radius-{none|xxs|xs|s|m|l|xl|2xl|3xl|4xl|full})

KEY COLOR TOKENS:
  Primary CTA / brand  →  var(--astral-color-brand-9)         ← green, NOT blue
  Page background      →  var(--astral-color-surface-default)
  Card background      →  var(--astral-color-surface-secondary)
  Primary text         →  var(--astral-color-text-default)
  Subtle text          →  var(--astral-color-text-subtle)
  Border               →  var(--astral-color-border-default)
  Hover state          →  var(--astral-color-surface-hover)

  Tight inline gap     →  var(--astral-units-pos-4)   = 4px

  Button padding (h)   →  var(--astral-units-pos-20)  = 20px
  Section gap          →  var(--astral-units-pos-24)  = 24px
  Large gap            →  var(--astral-units-pos-32)  = 32px

BORDER RADIUS CHEATSHEET:
  Badges / tags        →  var(--astral-border-radius-xxs)  = 4px

  Cards / dialogs      →  var(--astral-border-radius-m)    = 16px

  1. Layout  → use spacing tokens for all padding/gap/margin

  4. Radius  → pick the correct border-radius token for the shape
  5. Output  → complete production-ready code, no placeholders

Ready. Ask the designer: "What would you like to build?"`;
